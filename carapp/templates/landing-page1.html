About CarBnR

CarBnR is a full-stack web application that allows users to rent cars from private owners or list their own vehicles for rent. The platform includes user authentication, car search and filtering, booking management, and comprehensive dashboard functionality.

This project was built as a portfolio piece to demonstrate proficiency in modern web development technologies and practices using Python Flask and MySQL.

Key Features

Comprehensive functionality for both car owners and renters

Advanced Search Engine

Effortlessly find available rental cars with our powerful search engine. Filter by state, city, and location to quickly pinpoint your ideal vehicle. Features real-time availability, advanced filters (car type, price range, features), and an intuitive interface for fast, enjoyable searching.

Simplified Booking Process

Secure your chosen car with our streamlined booking form. Simply enter your pickup and return dates, we handle the rest. Enjoy instant booking confirmation and the flexibility to easily adjust dates as needed. Hassle-free booking has never been easier.

Smart Dashboard & Profile Management

Access a personalized dashboard that adapts to your role. Switch between renter and owner views. Owners can add unlimited locations (with name, address details) and vehicles (brand, model, daily price) at each location. Full CRUD operations allow editing or removing any location or vehicle.

Role-Based Access Control

Secure authentication system with distinct permissions for renters and owners. Owners exclusively confirm bookings for their vehicles, while renters can update or cancel their reservations within a 48-hour window. This ensures proper control and security for all users.

Comprehensive Vehicle Management

Owners can easily list their vehicles with detailed information, photos, and availability calendars. Add unlimited vehicles across multiple locations with full details including brand, model, daily price, and specifications. Comprehensive management tools allow for quick updates and viewing of booking history.

Location Management System

Owners can create and manage multiple rental locations with complete address details. Each location can host multiple vehicles, and owners have full control to add, edit, or remove locations as needed. This flexible system accommodates businesses of all sizes.

Development Highlights

Designed and implemented a normalized MySQL database with tables for users, locations, cars, bookings, and reviews. Established proper relationships between locations and vehicles for efficient data management.

Built a comprehensive REST API with Flask for user authentication, location management, car management, booking operations, and dashboard functionality. Implemented proper error handling and validation.

Implemented a dynamic dashboard that adapts based on user role (renter/owner). Owners can manage multiple locations and vehicles with full CRUD operations, while renters can track their bookings across different status categories.
